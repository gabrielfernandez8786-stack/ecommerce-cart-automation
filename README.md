# 🛒 Ecommerce Cart Automation

## 📌 Descripción
Framework de automatización de pruebas para el flujo de compra completo en **DemoBlaze** (tienda de demostración).

## ✅ Resultados
- ✅ **1/1 tests exitosos**
- ✅ CI/CD con GitHub Actions (pruebas automáticas en cada commit)

## 🛠️ Tecnologías
| Tecnología | Versión |
|------------|---------|
| Java | 17 |
| Selenium WebDriver | 4.x |
| TestNG | 7.11.0 |
| Maven | 4.x |
| WebDriverManager | 5.9.2 |
| GitHub Actions | CI/CD |

## 📋 Flujo automatizado
| Paso | Acción | Estado |
|------|--------|--------|
| 1 | Seleccionar producto "Samsung galaxy s6" | ✅ |
| 2 | Agregar producto al carrito | ✅ |
| 3 | Aceptar alerta de confirmación | ✅ |
| 4 | Navegar al carrito | ✅ |
| 5 | Validar producto en el carrito | ✅ |
